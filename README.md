# Arcanum Vivum

This repository is organized by language:

- `JavaScript` - original JavaScript spell engine library
- `C#` - Unity-ready C# port with matching core engine features
- `C++` - C++17 port with matching core spell engine features
- `GDScript` - Godot-ready GDScript port with matching core spell engine features

## Folder Layout

- `JavaScript/src/core` - feature extraction, embeddings, balancing, evolution, cast orchestration
- `JavaScript/src/audio` - audio capture and transcription integration
- `JavaScript/src/database` - in-memory, Firebase, Supabase, REST adapters
- `JavaScript/src/examples` - integration examples
- `C#/ArcanumVivum.Unity/Runtime` - C# Unity package runtime code
- `C#/ArcanumVivum.Unity/Samples~` - Unity sample script
- `C++/include` - C++ public API headers
- `C++/src` - C++ engine implementations
- `C++/examples` - C++ usage example
- `GDScript/addons/arcanum_vivum_spell_engine` - Godot addon-style GDScript implementation

Use each language folder independently.
